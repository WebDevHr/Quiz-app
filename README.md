# Todo-list with VueJs

It's a simple todo-list application.

## Run Locally

Clone the project

```bash
  git clone https://github.com/WebDevHr/E-commerce.git
```

Go to the project directory

```bash
  cd Todo
```

Install dependencies

```bash
  npm install
```

Start the application

```bash
  npm run
```
